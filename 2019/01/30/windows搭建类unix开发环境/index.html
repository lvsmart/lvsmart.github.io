windows搭建类unix开发环境

发表于 2019-01-30 | 更新于 2019-10-13 | 分类于 运维 | 评论数： | 阅读次数：

背景介绍

  • 运维开发的项目很多，其中有部分python和go的项目，依赖的类库不支持windows。（如kafka，ansible）
  • 日常工作依赖的企业微信，WPS办公软件，keepass等，离不开windows。
  • 综上，需要在windows系统中，打造类unix开发环境

系统要求

  • windows 10 最新版本
  • 开启WSL功能，并安装ubuntu，启动，设置用户名密码

使用软件

  • mobaxterm，免费版本即可

具体步骤

1、准备工作

  • mobaxterm会自动识别WSL安装的ubuntu,，创建一个默认配置的WSL session即可。
  • 使用国内ubuntu源，加速接下来的软件安装。常用的有：清华，阿里云，网易，中科大
  • 清华大学镜像站

2、安装vscode

  • 在官网下载ubuntu版最新deb包

  • 此时，窗口不能移动，不能全屏，还存在中文字体，以及中文输入法问题。（并非乱码，而是fonts问题）

  • 但是已经能够安装插件，代码高亮，跳转等正常开发功能了

3、安装xfce4

  • wsl默认使用ubuntu-server，没有桌面，安装一个轻量级桌面xfce4即可

  • 然后再新建一个图形化的session，具体配置如下图

  • 然后安装配置输入法（推荐fcitx + google拼音输入法）

4、开发环境

  • go直接使用源码

  • python系统自带python3（python3.6）

  • 一般需要额外安装gcc， python3-venv，python3-dev等一系列开发依赖包

5、最终效果

  • 项目代码可以直接使用windows下原有代码，配置好go path， venv等环境即可
